# 🥨샘미의 맛따라 멋따라🥨

### 프로젝트 소개
-----------
직접 가보거나 지인들에게 소개받은 50곳의 맛집리스트와 리뷰를 확인할 수 있습니다.


### 기능설명
-----------
main page
> 음식사진들이 정렬되어 있고 사진을 클릭했을 때 layerpopup을 이용해 리뷰를 확인할 수 있게 구현했습니다.

introduce page
> 웹페이지를 간단하게 소개하고 있습니다.
